# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-06

### Added
- Initial release
- Web video editor with React + TypeScript + Vite
- Backend video processing service with Node.js + Express + FFmpeg
- Shared package for common types and utilities
- Video upload and management
- Video trimming and merging
- Watermark support
- Video transcoding
- Job queue system with Bull and Redis
- Progress tracking
